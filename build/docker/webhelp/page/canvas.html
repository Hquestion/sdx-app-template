  • 输出预览

  • 组件日志

  • 工作流日志

Tab页签说明

画布中的Tab页签如下图所示，从左到右分别为：

  • 数据集/数据源：包括数据源、共享数据集和私有数据集。

    数据源用于存放平台文件系统数源和HDFS数据源；

    共享数据集存放平台中共享的公开数据集；

    私有数据集存放个人创建的私有数据集。

    注：

    此处不可新增，如要新增请参考“数据”模块的创建数据集。

  • 平台组件：用于存放平台提供的固定算法组件，可直接使用，不可编辑。

  • 自定义组件：用于存放当前用户创建的自定义算法组件。用户可以新增、编辑、删除、分享自定义算法组件。

  • 公共组件：用于存放其他人分享的自定义组件。

  • 收藏组件：用于存放用户收藏的自定义组件。

工具菜单说明

工作流工具菜单如下图所示，从左到右分别为：

  • 保存：保存当前画布中的实例。

  • 另存为：将当前画布中的实例另存为另一工作流。

  • 撤销：撤销画布中上一步操作。

  • 恢复：恢复画布中上一步操作。

  • 运行/停止：手动运行或停止画布上当前实例。

  • 定时运行：设置画布定时运行的周期和时间，设置定时运行后，每次定时运行后会在定时运行页面产生一条定时运行记录。

  • 执行设置：包括并行和顺序执行。并行执行表示多个分支下，多分支同时执行；顺序执行表示在多个分支的情况下，先执行完第一个分支的所有节点再执行第二个分支。

  • 清除：清除当前画布中的实例。

  • 比例缩放：按比例对当前画布进行缩放。

组件常见状态说明

组件常见的几种状态如下图所示。

  • 初始：组件配置成功，已在后台生成实例。

  • 参数配置异常：组件未配置相关参数或参数配置有误。

  • 运行中：组件运行中。

  • 错误：组件运行失败。
