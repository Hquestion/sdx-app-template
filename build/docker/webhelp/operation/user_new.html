你可以创建一个用户并让其加入用户组

前提条件

当前系统中已存在创建好的角色。

操作步骤

  1. 以管理员用户登录平台，单击左侧导航栏的用户管理 > 用户。

    系统展示用户窗口。

  2. 配置新增用户信息，如下图所示。

    相关参数描述如下：

    • 用户名：新增的用户名，自定义，在文本框中输入1–64位字符

    • 显示名：用户的显示名，自定义，在文本框中输入

    • 密码：新增用户的密码，在文本框中输入

    • 角色：新增用户的角色，在下拉列表框中选择

    • 是否激活：选择“是”，表示直接激活该用户；选择“否”，表示当前不激活该用户，该用户无法正常登录平台

    • 有效期：在下拉列表框中选择该用户的有效期

  3. 配置完成后，单击确定。

    新建的用户展示在用户列表中，如下图所示。

加入用户组

  1. 单击用户列表后的加入组小图标。

    系统展示加入组设置窗口。

  2. 在左侧选择需要加入的用户组移动至右侧，如下图所示。

  3. 单击确定。
