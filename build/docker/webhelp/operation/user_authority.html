你可以对用户进行管理平台模块的授权

此操作为授权对象添加权限设置中配置的相关权限。

背景信息

授权页面包括：用户授权列表、用户组授权列表和角色授权列表三个页签，支持分别为用户、用户组和角色进行授权。

用户、用户组和角色的授权方式相同，本文仅以新建用户授权为例进行说明。

新建用户授权

  1. 以管理员用户登录系统后，单击左侧导航栏的权限管理 > 授权。

    系统展示授权页面，选择用户授权列表页签。

  2. 单击新建授权按钮。

    系统弹出新增授权窗口。

  3. 配置权限信息。

    • 授权对象：在文本框中输入

      • 新建用户授权时输入需要授权的用户名

      • 新建用户组授权时输入需要授权的用户组名

      • 新建角色授权时输入需要授权的角色名

    • 权限设置：左侧展示系统中所有权限，支持搜索，支持勾选相关的权限移至右侧，也支持全部移动至右侧

  4. 配置完成后，单击确定。
