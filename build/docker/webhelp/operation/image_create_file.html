你可以基于文件构建一个镜像

基于文件包括如下两种方式构建，你可以选择其中一种进行构建镜像。

基于tar文件构建

  1. 登录平台后，单击左侧导航栏的资产管理 > 镜像管理。

    系统展示镜像列表页签。

  2. 单击镜像列表页面的基于文件构建按钮。

    系统展示基于文件构建页面。

  3. 按下图所示操作构建镜像。

基于DockerFile文件构建

  1. 登录平台后，单击左侧导航栏的资产管理 > 镜像管理。

    系统展示镜像列表页签。

  2. 单击镜像列表页面的基于文件构建按钮。

    系统展示基于文件构建页面。
